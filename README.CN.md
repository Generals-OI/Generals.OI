# Generals.OI

Generals.OI 是正在开发的 <https://generals.io> 的局域网联机版，旨在给玩家提供更稳定流畅的游戏体验~~和更丰富的内容~~。

![](https://raw.githubusercontent.com/gfy1729/MyPictures/main/G-1.0/sample2.png)

项目使用 Qt 6 开发，兼容 Qt 5 环境。
